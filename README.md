# Proyecto-SunnyDream2026
Proyecto Semestral para Desarollo FullStack 2026 DuocUc
Descripción del proyecto:
SunnyDream es una tienda de accesorios para mujer enfocada principalmente en la venta de coles para el cabello y popsockets para celulares.
El proyecto busca modernizar la gestión de la tienda mediante un sistema que permita administrar productos, ventas, stock, clientes y usuarios de forma más ordenada y eficiente.

El sistema fue desarrollado utilizando una arquitectura basada en microservicios, donde cada módulo cumple una función específica y cuenta con su propia base de datos.

Objetivo del sistema

Crear un sistema de ventas que permita gestionar los principales procesos de SunnyDream, mejorando el control de productos, inventario y ventas.
Además, el sistema busca facilitar el crecimiento futuro del negocio mediante una arquitectura modular y escalable.
